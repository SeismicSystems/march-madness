### Contract Integration
You'll need the MarchMadness ABI. You can define it inline or import from the client package. Key calls:
- `submitBracket(bytes8)` — shielded write via walletClient.writeContract(), payable 1 ETH
- `updateBracket(bytes8)` — shielded write
- `setTag(string)` — regular write
- `getBracket(address)` — signed read (own, before deadline) or transparent read (anyone, after deadline)
- `getEntryCount()` — transparent read
- `getTag(address)` — transparent read

### Important Architecture Note
The contract address should come from an env var: `VITE_CONTRACT_ADDRESS`
