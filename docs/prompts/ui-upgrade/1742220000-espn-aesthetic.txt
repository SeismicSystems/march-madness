# UI Upgrade: ESPN-like Sports Aesthetic

## Context

The current UI uses Fira Mono (monospace), muted mauve accents, and flat dark grays — it reads as a developer tool, not a sports app. The goal is to make it feel like ESPN: bold, glossy, energetic, and unmistakably "sports". This is a purely visual overhaul — no functionality changes.

## Approach: Theme-first, then components

### Step 1: Foundation — Fonts + Color Palette

**Files:** `packages/web/index.html`, `packages/web/src/index.css`

**Fonts:**

- **Inter** (body/UI) — clean modern sans-serif, great for data-heavy sports UIs
- **Oswald** (display/headers) — condensed bold, very ESPN for section titles like "FINAL FOUR", region names, countdown numbers
- Keep **Fira Mono** only for hex values, addresses, scores (`font-mono` class)

**New color palette (replace entire `@theme` block):**

- Backgrounds: navy-blacks (`#0a0e1a`, `#111827`, `#1e2a3a`) instead of flat grays
- Text: clean whites/cool grays (`#f1f5f9`, `#94a3b8`) instead of sepia tones
- Accent: vibrant orange (`#f97316`) instead of muted mauve
- New: region colors (blue/red/green/purple for East/West/South/Midwest)
- New: gold/silver/bronze for leaderboard podium
- New: page gradient background (navy → deep purple-navy at 135°)

**Add custom utility classes:**

- `.bg-page` — diagonal gradient background for the app shell
- `.text-display` — Oswald font family shortcut

> This single step transforms ~60% of the visual feel since every component uses these CSS variables.

## Key Design Decisions

- **Dark theme stays** — sports apps look great dark, and it matches the existing brand
- **Orange accent** — ESPN's energy, universally "sports", high contrast on navy
- **Inter + Oswald combo** — proven sports typography pairing (Inter for readability, Oswald for impact)
- **Region color coding** — map region name → color inside BracketRegion.tsx (no prop changes needed)
- **No layout changes** — all responsive breakpoints and component structure stay the same
