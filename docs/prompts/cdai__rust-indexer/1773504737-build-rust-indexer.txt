## What You Need to Build

### Core Indexer (`src/main.rs` + modules)

**1. Event Listener**
- Connect to a Seismic node via JSON-RPC (WebSocket or HTTP, configurable)
- Subscribe to / poll for `BracketSubmitted` and `TagSet` events from the MarchMadness contract
- On `BracketSubmitted`: add/update entry in the index with address, block number, timestamp
- On `TagSet`: update the entry's name field
- Write the index to a JSON file after each update

**2. Backfill Command**
- `indexer backfill` — scan all historical blocks for relevant events and rebuild the index from scratch
- Should handle pagination for large block ranges (batch `eth_getLogs` calls)
- After backfill, do a sanity check: compare index entry count with contract's `getEntryCount()`

**3. Bracket Reveal (post-deadline)**
- `indexer reveal` — after the submission deadline, call `getBracket(address)` for each indexed address to read their bracket
- Update the index with the bracket hex data
- This only works after the deadline when brackets become publicly readable

**4. Sanity Check Command**
- `indexer check` — compare the number of entries in the local index with the contract's `getEntryCount()` return value
- Report any discrepancy

**5. File Locking / Mutex**
- Use file-based locking (e.g., `fs2` crate or advisory locks) to prevent simultaneous writes to the index file
- The server will be reading this file concurrently

### JSON Index File Format
The file is a JSON object keyed by lowercase hex address:
```json
{
  "0x1234...abcd": {
    "name": "optional-tag",
    "updated": {
      "block": 12345,
      "ts": 1741945200
    },
    "bracket": "0x8000000000000001"
  }
}
```
- `name` is omitted if no tag set
- `bracket` is omitted until after reveal
- Use the types from `march_madness_common`

### Ethereum ABI Encoding
For decoding event logs and making contract calls, you can use:
- `alloy-primitives` for types (Address, B256, etc.)
- `alloy-sol-types` for ABI encoding/decoding
- Or do manual ABI encoding since the interface is simple

### Event Topics
- `BracketSubmitted(address indexed account)` — topic0 = keccak256("BracketSubmitted(address)")
- `TagSet(address indexed account, string tag)` — topic0 = keccak256("TagSet(address,string)"), topic1 = address, data = ABI-encoded string

### Contract Calls
- `getEntryCount()` — selector = first 4 bytes of keccak256("getEntryCount()")
- `getBracket(address)` — selector = first 4 bytes of keccak256("getBracket(address)"), returns bytes8
