# Plan: Add NCAA Team Logos to Bracket UI

## Context

The bracket UI currently displays teams as text-only (seed + name). We want to add team logos next to each team name, positioned on the **outer edge** — left side for left-side regions (East, South), right side for right-side regions (West, Midwest). This makes the bracket feel more polished and visually scannable.

The NCAA API does not provide logo URLs, so we'll use the **ESPN CDN** (`https://a.espncdn.com/i/teamlogos/ncaa/500/{espn_id}.png`), which is the standard approach used across sports apps.

## Implementation Steps
